# minemanager
## A Minecraft Server management tool.
**Features:**
1. Create servers
2. Import/Export servers
3. Import/Export worlds
4. Rename servers
5. Delete servers
6. Delete worlds
7. Update server.properties file from template
8. Unban all players and ips
9. Deop all players
10. Automatically accept EULA
11. Automatically make start.sh/start.bat script
